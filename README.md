# 🪨 Эволюция Гальки

Кликер / Idle игра для платформы Яндекс.Игры.

## Описание

Пройди путь от маленькой Гальки до величественной Скалы! Кликай, покупай улучшения, зарабатывай пассивный доход с помощью Ворона и защищайся от случайных событий.

## Особенности

- **12 уровней эволюции** — от Гальки до Скалы
- **Пассивный доход** — купленный Ворон приносит очки каждую секунду
- **Случайные события** — гроза, ветер, дождь, охотник, метеорит
- **Защитные навыки** — Фортуна и Крепость камня
- **Yandex SDK** — облачные сохранения, реклама, лидерборды

## Стек

- **TypeScript** (strict mode)
- **Phaser 3** — игровой движок
- **Vite** — сборка и dev-сервер
- **Yandex Games SDK** — интеграция с платформой

## Установка

```bash
# Установите Node.js (>= 18)
# Установите зависимости
npm install

# Запуск dev-сервера
npm run dev

# Production сборка
npm run build
```

| Команда | Описание |

## Структура проекта

```
src/
├── core/           # StateManager, config, типы
├── modules/        # Игровые системы (клик, Ворон, события)
├── ui/             # UI-компоненты
├── scenes/         # Phaser сцены (Boot, Game, Shop)
├── assets/         # Графика, звук
├── sdk/            # Обертки Yandex SDK
└── main.ts         # Entry point
```

## Архитектура

Игра построена по принципу **модульной архитектуры** с централизованным `StateManager`, который управляет состоянием через реактивные подписки (EventEmitter). Каждый игровой модуль независим и взаимодействует через события.
